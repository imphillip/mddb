# Contributor Covenant Code of Conduct

## Our pledge

We want mddb.dev to be a useful public model registry maintained through careful, evidence-based collaboration. We pledge to make participation in this project respectful and harassment-free for everyone.

## Expected behavior

- Be respectful and constructive.
- Argue from evidence, source URLs, tests, and reproducible examples.
- Assume good faith, but be precise about data-quality risks.
- Keep discussions focused on the registry, code, documentation, and review criteria.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without permission.
- Spamming issues or pull requests.
- Submitting fabricated source evidence or intentionally misleading model data.

## Enforcement

Maintainers may edit, hide, reject, or remove comments, issues, pull requests, or contributions that violate this code of conduct. Repeated or severe violations may lead to a ban from project spaces.

## Reporting

Open a GitHub issue or contact the maintainers privately if public reporting is not appropriate.
